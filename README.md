# rockgame
Stone-Paper-Scissor Game Using HTML,CSS,JS and Django ( BASIC LEVEL )

# FEATURES (Phase 1)
1. User login/register (Django auth system)
2. Home page:
-> Play Online (blank for now)
-> Play With Computer

3. If clicked Play With Computer:
-> Play for 10 points
-> Play Unlimited

4. Game page:

-> Rock / Paper / Scissors buttons
-> Score display
-> Result display

5. JS handles:
-> Computer random choice
-> Winner logic
-> Score tracking
-> Game over (if 10 points mode)

6. Django handles:
-> Authentication
